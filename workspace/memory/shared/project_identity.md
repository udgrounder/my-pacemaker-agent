# 프로젝트 정체성

## 한 줄 정의

AI agent와의 협업 구조를 파일 시스템으로 인코딩한 재사용 가능한 워크플로우 패키지.

## 무엇을 하는가

대상 프로젝트에 설치되어 AI agent가 계획 수립 → 검토 → 승인 → 구현 → 결과 검토의 구조화된 흐름으로 작동하도록 한다. 에이전트의 자율적 판단이 필요한 지점과 사용자 결정이 필요한 지점을 명확히 분리하는 것이 핵심이다.

## 구성

| 구성 요소 | 위치 | 역할 |
|----------|------|------|
| 배포 파일 | `dist/` | 대상 프로젝트에 설치되는 실제 시스템 |
| 에이전트 명세 | `agent-specs/` | claude / codex / antigravity 별 설치 설정 |
| 설치 스크립트 | `install.py` | 신규 설치 및 업그레이드 처리 |
| 가이드북 | `guidebook/guidebook.md` | 방법론 전체 문서 |

## 사용 대상

AI agent(Claude Code 등)와 함께 소프트웨어를 개발하는 프로젝트.  
설치 후 대상 프로젝트의 `.mpa-workspace/`와 `workspace/`에 체계가 적용된다.
