# 선택적 확장 (MCP)

하네스 자체는 경량으로 유지된다. 프로젝트 규모나 필요에 따라 아래 MCP 서버를 추가로 연결한다.

---

## 코드 인덱싱

**용도:** 대형 코드베이스에서 의미 기반 파일 탐색, 심볼 검색, 파일 간 참조 관계 조회

**연결 방법 (Claude Code 기준):**

`.claude/settings.json` 또는 프로젝트 루트 `CLAUDE.md`에 MCP 서버 설정 추가.

```json
{
  "mcpServers": {
    "code-index": {
      "command": "...",
      "args": ["..."]
    }
  }
}
```

**MCP 연결 후 agent가 할 수 있는 것:**
- 자연어로 관련 파일 탐색: "결제 처리 관련 파일"
- 심볼 위치 조회: "PaymentService 클래스가 어디 있나"
- 참조 관계 추적: "이 함수를 호출하는 곳 전체"

**소규모 프로젝트:** MCP 없이 코드 맵 + grep으로 충분.  
**대형 프로젝트:** MCP 코드 인덱싱이 실질적으로 필요.

---

## 확장 추가 시

새 MCP 확장을 팀에 도입할 때:
1. 이 파일에 용도와 연결 방법 추가
2. `agent_rules.md` 의 관련 섹션에 사용 시점 명시
