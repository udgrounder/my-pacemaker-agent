# 메모리 인덱스

> **이 인덱스는 선별용 캐시다. 정답은 항상 `domains/`·`roles/` 폴더 본체다.**
>
> - **용도:** 작업 라우팅 시 이 파일을 먼저 읽어, 현재 작업에 **매치되는 메모리만** 골라 로드한다. (읽기 전 선별)
> - **제외:** `shared/`(architecture·contracts·direction·project_identity)는 라우팅 표가 고정 이름으로 직접 로드하므로 여기에 등록하지 않는다.
> - **동기화 원칙:** 폴더에 파일이 있는데 INDEX에 없는 항목만 누락 처리. 메모리 파일을 생성·갱신할 때 해당 행을 등록·갱신한다. INDEX가 본체와 어긋나면 폴더를 신뢰한다.
