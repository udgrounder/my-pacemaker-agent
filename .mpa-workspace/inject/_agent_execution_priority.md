# 에이전트 실행 우선순위

> **이 섹션을 참조하는 파일:** `layer1_critique.md`, `layer1_review.md`
>
> 컨텍스트 격리가 필요한 세션(비평·검증)의 실행 방식을 결정한다.
> 에이전트가 제공하는 기능 범위 안에서 아래 순서대로 실행한다.

| 우선순위 | 조건 | 실행 방법 |
|---------|------|---------|
| 1순위 | Agent 도구(서브에이전트) 사용 가능 | **서브에이전트로 실행** — 컨텍스트 물리적 격리 |
| 2순위 | Agent 도구 없음 | **자가 비평/검증으로 대체** — 아래 제약 준수 필수 |

**2순위 실행 시 필수 제약:**

```
⚠️ 서브에이전트를 사용할 수 없어 같은 스레드에서 진행합니다.
   잡을 수 없는 것: 설계자/구현자 본인의 숨은 전제, 관점 편향.
   위험도가 높은 작업이면 새 세션에서 직접 비평/검증하는 것을 권장합니다.
```

제약 (준수하지 않으면 무의미):
1. **대상 파일만 읽는다** — 설계/구현 대화 컨텍스트를 참조하지 않는다
2. **"실패 증거를 찾는다"는 자세** — "좋아 보인다"는 답을 허용하지 않는다
3. **파일에서 직접 인용한 근거**를 각 항목에 포함한다
