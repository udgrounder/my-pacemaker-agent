# 🛠️ AI 에이전트 실전 활용 사례 (Use Cases)

이 디렉토리는 사용자님이 일상 업무, 프로젝트 기획, 프로그래밍 등 다양한 도메인에서 **AI 에이전트를 실제로 어떻게 활용하여 문제를 해결했는지** 그 과정과 결과를 기록하는 케이스 스터디 보관소입니다.

단순히 "결과가 잘 나왔다"를 넘어 **"어떤 방식으로 소통하고 조율했는지(워크플로우)"**에 집중해 기록을 누적하면, 나중에 자신만의 아주 강력한 'AI 협업 매뉴얼'이 됩니다.

---

## 📝 사례 작성 템플릿 (Template)

새로운 활용 사례를 기록할 때는 아래의 템플릿을 복사하여 `case_01_주제명.md` 형태로 파일을 만들어 작성해 보세요.

```markdown
# [Case 0X] 사례 제목 (예: AI 에이전트와 함께한 Next.js 프로젝트 세팅)

* **진행 일자**: 2026-0X-0X
* **대상 에이전트**: Antigravity, Cursor, ChatGPT 등
* **목표**: 해결하고자 했던 핵심 문제나 목표 기재

## 1. 협업 프로세스 (Workflow)
* 에이전트에게 내린 첫 번째 의도(프롬프트)와 역할 부여.
* 에이전트가 제안한 계획(Plan) 중 마음에 들었던 점과 수정했던 점.

## 2. 해결 과정 및 결과
* 실제로 생성/수정된 주요 코드나 문서 링크.
* 어떤 식으로 완성도를 검증(Verify)했는지 기재.

## 3. 회고 (Lessons Learned)
* **잘된 점**: 에이전트를 사람처럼 잘 다루어 효율을 극대화한 순간.
* **아쉬운 점 / 한계**: 소통의 오류가 났거나 에이전트가 헤맸던 부분과 그 대처법.
* **앞으로의 팁**: 다음번에 유사한 작업을 할 때 기억할 마인드셋 팁.
```
